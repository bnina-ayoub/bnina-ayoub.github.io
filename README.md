## ▍ What this repository is

This repo hosts my personal digital portfolio — a living dossier that documents the embedded systems, robotics and defense-tech work I've spent the last several years building. It is the canonical place I send recruiters, professors, collaborators and curious engineers to see *what I do* and *how I think*.

## ▍ Featured target profiles

**`TGT-001` — Anti-UAV Target Detection & Tracking** *— ongoing*
Custom YOLOX vision architecture with an early-exit mechanism for ultra-low-latency target detection, paired with NXP and STM32 embedded hardware for real-time sensor processing and countermeasure actuation.

**`TGT-002` — PID Pan-Tilt Aircraft Tracker** *— completed*
Real-time aircraft tracker built on ThreadX RTOS and STM32 timer peripherals driving precise servo actuation via PWM, with a micro-ROS / ROS 2 geometry pipeline for low-latency host communication.

**`TGT-003` — Quadruped Locomotion Stack** *— archive*
Embedded gait controller and DDS bridge for a four-legged robot — STM32 talking to a Raspberry Pi planner over micro-ROS.

**`TGT-004` — Underwater AUV Control Node** *— archive*
ThreadX-based control node on STM32U5, bridged to an Nvidia Jetson companion computer for an underwater autonomous platform.

## ▍ Design doctrine

The visual identity was a deliberate choice — equal parts **industrial robotics blueprint** and **tactical defense HUD**. Graphite at the base. Signal orange for what matters. HUD green for what's alive. Tactical amber for what's incoming. Monospace for anything an oscilloscope would output. Scanlines, blueprint grid and the occasional radar sweep — because the work itself looks like that on the bench.

Built to feel like a console an operator would actually use, not a brochure a marketer would design.

---

## ▍ Engineering values

- **Determinism over magic.** If it isn't predictable in microseconds, it doesn't ship.
- **Bare-metal first.** Abstractions earn their place by saving cycles or saving sanity — preferably both.
- **The bus is the contract.** UART, SPI, DDS, ROS 2 — protocols are honored to the bit.
- **Document like the next operator is you at 3am.** Because eventually it is.
